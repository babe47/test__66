# MQTT 監控儀表板 - 產品需求文件 (PRD)

## 專案概述
建立一個基於 Streamlit 的即時監控儀表板,用於顯示透過 MQTT 協議接收的智慧家居感測器數據。

## 系統角色
- **角色定位**: MQTT 訂閱者 (Subscriber)
- **應用程式名稱**: `app.py`

---

## 功能需求

### 1. 資料顯示
儀表板需要即時顯示以下資訊:

#### 1.1 電燈狀態
- 顯示電燈的開關狀態
- 使用視覺化指示器(如圖標或顏色)清楚標示開啟/關閉狀態

#### 1.2 溫度監控
- 顯示客廳當前溫度
- 以圖表形式呈現溫度變化趨勢
- X 軸:日期與時間
- Y 軸:溫度值(°C)

#### 1.3 濕度監控
- 顯示客廳當前濕度
- 以圖表形式呈現濕度變化趨勢
- X 軸:日期與時間
- Y 軸:濕度值(%)

### 2. 資料儲存
- 將所有接收到的 MQTT 資料自動儲存為 Excel 檔案
- 檔案格式:`.xlsx`
- 儲存欄位應包含:時間戳記、資料類型、數值

## 技術規格

### 開發環境
- **程式語言**: Python
- **主要框架**: Streamlit
- **通訊協議**: MQTT
- **虛擬環境位置**: `/home/pi/Documents/Github/test__66/.venv`

### 功能限制
- 本應用程式為**唯讀模式**,僅提供資料顯示功能
- 不包含控制功能(如開關電燈、調整溫度等)

---

## 使用者介面設計原則
1. 介面簡潔直觀,資訊一目了然
2. 使用即時更新機制,確保資料的時效性
3. 圖表需清晰易讀,支援時間範圍篩選
4. 響應式設計,適配不同螢幕尺寸
